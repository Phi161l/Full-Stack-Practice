# Fullstack Practice
A personal repository for experimenting and learning full-stack development.  
Contains small projects and mini-experiments across **frontend, backend, databases, Node.js, Express, and more**.  
Each project is self-contained and designed to practice new concepts or test out ideas.

## Features & Learning Goals
- **Frontend:** HTML, CSS, JavaScript, React, Next.js, or other frameworks  
- **Backend:** Node.js, Express.js, REST APIs, authentication  
- **Databases:** SQL (PostgreSQL, MySQL), NoSQL (MongoDB)  
- **Fullstack integration:** Connecting frontend to backend and database  
- **Experimentation:** Trying out new libraries, frameworks, and tools

To explore a specific concept:

cd project-name
# Follow the README inside that project (if available)
